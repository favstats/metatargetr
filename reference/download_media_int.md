# Download media files with specified IDs

Download media files with specified IDs

## Usage

``` r
download_media_int(id, x, n, mediadir = "data/media")
```

## Arguments

- id:

  Ad ID.

- x:

  Media URLs to download.

- n:

  Number of URLs.

- mediadir:

  Directory to save media files.

## Value

A character vector with file paths.
