# Walk through a list with a progress bar

Walk through a list with a progress bar

## Usage

``` r
walk_progress(.x, .f, ...)
```

## Arguments

- .x:

  List to iterate over.

- .f:

  Function to apply.

- ...:

  Other parameters passed to
  [`purrr::map_dfr`](https://purrr.tidyverse.org/reference/map_dfr.html).

## Value

None (used for side effects).
